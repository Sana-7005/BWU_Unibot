# 🤖 Unibot

An intelligent AI-based chatbot that combines document retrieval and language models to deliver accurate, context-aware university information.

---

## 📌 Project Overview

Unibot is an AI-driven chatbot designed to assist students by answering university-related queries in natural language.

The chatbot handles questions related to:
- 🎓 Scholarships  
- 📅 Campus events  
- 💼 Placements  
- 📚 Academic information  

👉 Unlike traditional chatbots, Unibot uses a *retrieval-based approach*:
- First retrieves relevant data from official documents  
- Then generates accurate responses  

This ensures higher reliability and reduces incorrect answers.

---

## ❗ Problem Statement

In universities, important information is scattered across:
- Notices  
- PDFs  
- Internal data files  

Students face difficulty finding accurate information quickly.

### ⚠️ Challenges:
- Keyword-based systems lack understanding  
- Generic AI chatbots may give incorrect answers  
- No centralized information system  

👉 This project solves these issues using *document retrieval + AI response generation*

## 🤖 Artificial Intelligence Integration

Unibot integrates multiple AI components:

| AI Function | Purpose |
|------------|--------|
| Text Embedding Model | Converts text into vectors |
| Vector Search Engine | Finds relevant documents |
| Contextual Retrieval | Fetches supporting information |
| Query Classification | Identifies query type |
| Language Model | Generates final response |

👉 Ensures responses are based on verified data, not assumptions
